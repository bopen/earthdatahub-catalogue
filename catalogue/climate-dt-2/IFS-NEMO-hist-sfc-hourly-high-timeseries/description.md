This dataset contains surface variables from the second generation Climate Adaptation Digital Twin of the Destination Earth initiative. It includes data from the Historical Run (1990–2014) of the IFS-NEMO model. This data was regridded from the original HEALPix H1024 grid to a regular latitude-longitude grid at approximately 0.044° resolution. The temporal resolution is 1 hour.

The dataset's chunking is optimized for regional timeseries analisys.

Accessing this dataset requires a Zarr v3-compatible library (e.g., zarr-python >= 3.x). Python users can upgrade via `pip install "zarr>3"`.

Access is restricted to Destination Earth users with Upgraded Access.
